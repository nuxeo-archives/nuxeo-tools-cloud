Those scripts create images with Nuxeo 6.0 LTS using PostgreSQL and an Apache front-end in different cloud platforms.
The base OS is Ubuntu 14.04 LTS.

Requirements:

- AWS: you must have the aws CLI installed and your Access Key ID and Secret Access Key in your environment.

- Google Compute Engine: you must have the gcloud CLI installed and configured for your account.

- Azure: you must have the azure CLI installed and configure for your account in Service Management mode (azure config mode asm).
